# Paleta Oficial Anclora Nexus Group (Dark Mode)

Esta paleta define los colores institucionales para la marca madre Anclora Nexus Group, optimizada para interfaces de alta fidelidad y lujo.

## Colores Principales

| Color | HEX | RGB | Uso Principal |
| :--- | :--- | :--- | :--- |
| **Anclora Gold** | `#C5A059` | `197, 160, 89` | Acentos, Botones Primarios, Iconos destacados. |
| **Anclora Navy** | `#05070A` | `5, 7, 10` | Fondo principal, Capas base. |
| **Dark Slate** | `#0A0F14` | `10, 15, 20` | Fondos de secciones, Cards, Degradados. |
| **Crystal White** | `#FFFFFF` | `255, 255, 255` | Texto principal, Títulos. |

## Colores Secundarios y Semánticos

| Aplicación | Valor | Opacidad |
| :--- | :--- | :--- |
| **Texto Secundario** | `#FFFFFF` | `0.7` |
| **Bordes Sutiles** | `#C5A059` | `0.1` |
| **Overlay / Vidrio** | `rgba(0, 0, 0, 0.4)` | `-` |
| **Error (Destructive)** | `#FF4B4B` | `-` |

## Reglas de Uso
1. **Contraste**: El texto Crystal White debe usarse siempre sobre Anclora Navy o Dark Slate.
2. **Jerarquía**: El Gold se reserva para elementos que requieren atención inmediata (Call to Action).
3. **Elevación**: Usar variaciones de Dark Slate para indicar elevación en cards sobre el fondo Anclora Navy.
