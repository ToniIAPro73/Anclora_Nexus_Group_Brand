# Paleta de Colores - Anclora Cognitive Solutions

División tecnológica y de análisis de datos. Utiliza una variante de la paleta clásica de Anclora con un enfoque en el contraste digital y la sofisticación técnica.

## Colores Principales

| Color | HEX | RGB | Uso |
| :--- | :--- | :--- | :--- |
| **Cognitive Gold** | `#C5A059` | `197, 160, 89` | Acentos técnicos, data visualization highlights. |
| **Cognitive Black** | `#05070A` | `5, 7, 10` | Fondo de aplicaciones (Deep Black), máxima profundidad. |
| **Data Slate** | `#0A0F14` | `10, 15, 20` | Superficies de cards, sidebars, layers adicionales. |
| **Pure White** | `#FFFFFF` | `255, 255, 255` | Texto principal, máxima legibilidad sobre fondo negro. |

## Estética sugerida
- **Glow Effects**: Uso de sombras externas con 15% de opacidad del Cognitive Gold para elementos activos.
- **Micro-interacciones**: Transiciones rápidas y precisas.
- **Glassmorphism**: Backdrop filters con desenfoque de 20px-30px para modales.

## Tipografía recomendada
- **Global**: Montserrat para un look tecnológico y limpio.
- **Numérico**: Inter o Roboto Mono para tablas de datos y métricas.
