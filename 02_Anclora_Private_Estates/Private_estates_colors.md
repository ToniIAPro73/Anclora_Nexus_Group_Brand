# Paleta de Colores - Anclora Private Estates

Esta paleta está diseñada para la división de Real Estate de lujo, utilizando tonos dorados clásicos y un azul marino profundo e institucional.

## Colores Principales

| Color | HEX | RGB | Uso |
| :--- | :--- | :--- | :--- |
| **Estate Gold** | `#D4AF37` | `212, 175, 55` | Logos, acentos, botones de acción. |
| **Estate Navy** | `#0A1F3D` | `10, 31, 61` | Fondos institucionales, cabeceras. |
| **Champagne** | `#E6C96E` | `230, 201, 110` | Variación clara para hovers o textos destacados. |
| **Bronze Gold** | `#B9915F` | `185, 145, 95` | Variación oscura para bordes o sombras suaves. |

## Tipografía recomendada
- **Títulos**: Playfair Display (Serif) para un toque clásico de lujo.
- **Cuerpo/UI**: Outfit o Montserrat (Sans-serif) para legibilidad moderna.

## Reglas de Uso
- El dorado `#D4AF37` debe usarse con moderación para mantener la elegibilidad y el aire de exclusividad.
- Combinar con blanco puro o gris muy claro para superficies de lectura.
